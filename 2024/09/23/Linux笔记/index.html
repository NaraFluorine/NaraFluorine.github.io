使用

Putty登录简述

管理员

root,根用户.拥有最高权限.
普通用户的提示符是 $ ,管理员的命令提示符是 # .(这个可以设置,丰富系统信息)

命令

格式是 $ 命令 [-选项] [参数] 表示三个字段.

简单的命令手册

passwd :修改密码.
passwd username :修改别人密码.(显然,只有管理员可以)(root用户创建新用户时分配初始口令)
exit :退出系统.
logout :退出系统.
[Ctrl+d] :退出系统.
date :系统告诉你当前时间.
