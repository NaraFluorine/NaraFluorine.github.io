Java

别问,问就是规定.

Java:外部类的名字和文件名必须一致,而且首字母必须大写.
即一个文件中可以有很多 class ,但是这些类都是内部类,出了文件就不能用,除了唯一可以用 public 修饰的和文件名相同的类.

final

就是const和constexpr…

一个新自定义类

永远是 xx x=new xx() .

public , protected & private

所有变量,函数,哪怕构造函数都要有一个这样的声明,默认是 private .

权限和c++一样.

static

可以用于子类,即class里面的class可以修饰.

main接口必须是静态的,也就是 public static void main(String[] args) 简直buff叠满.

extends

继承.一个类只能继承一个类,但是可以套娃继承.

如果一个类被声明为final则不可继承,方法被声明为final则不可被覆盖
