发现 $63594$ 这个数字有一个大质数因数,所以答案更有可能是加出来的,而不是乘出来的,也就是说可能有多种情况.

直接爆搜完全对称的能得到 63504 ,后续发现 2 0 0 8 和 0 2 8 0 这种也是可以的,加进去得到了小样例的答案.

至此我们有了一个大致思路:分别枚举多少个子是什么情况,然后都乘起来即可,复杂度O(n^3).
但这显然会超时,考虑优化:
