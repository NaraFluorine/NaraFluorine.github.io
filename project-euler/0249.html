deepseek v4 thinking

以下是解决该问题的 C++ 代码。它使用动态规划（0/1 背包）计算所有子集的和，统计和为素数的子集个数，并输出最终答案的后十六位。

算法要点：

  • 用埃拉托色尼筛法生成小于 5000 的素数集合 S，并求出其总和 sum_all。
  • 再筛一次得到 [2, sum_all] 范围内的素数标记，用于最后判断。
  • dp 数组记录每种和值的子集计数，采用倒序更新实现 0/1 背包。
  • 取模使用条件减法 if (dp[s] >= MOD) dp[s] -= MOD; 比 % 运算更高效。
  • 时间复杂度约 6.7 亿次加法，现代 CPU 可在数秒内完成。
