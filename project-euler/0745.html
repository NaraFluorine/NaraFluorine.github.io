f是无平方因子数的个数

好的，这道题的 $N = 10^{14}$，所以 $\lfloor\sqrt{N}\rfloor = 10^7$。

我们来分析一下 $S(N)$ 的计算：
$S(N)=\sum_{i=1}^{\lfloor\sqrt{N}\rfloor} i^2 \cdot f\left(\left\lfloor\dfrac{N}{i^2}\right\rfloor\right)$

我们称 $S_{outer} = \lfloor\sqrt{N}\rfloor = 10^7$。
我们需要一个外层循环 $i$ 从 $1$ 遍历到 $S_{outer}$。

在循环内部，我们需要计算 $f(M)$，其中 $M = \lfloor N/i^2 \rfloor$。
根据定义，$f(M) = \sum_{k=1}^{\lfloor\sqrt{M}\rfloor} \lfloor \frac{M}{k^2} \rfloor \mu(k)$。

我们称 $S_{inner} = \lfloor\sqrt{M}\rfloor$。
我们需要一个内层循环 $k$ 从 $1$ 遍历到 $S_{inner}$。

我们需要 $\mu(k)$ 的值。$k$ 的最大取值是多少？
$k$ 的最大值取决于 $S_{inner}$ 的最大值。
$S_{inner} = \lfloor\sqrt{M}\rfloor = \lfloor\sqrt{\lfloor N/i^2 \rfloor}\rfloor$。
当 $i=1$ 时，$M$ 取最大值 $N$，$S_{inner}$ 取最大值 $\lfloor\sqrt{N}\rfloor = 10^7$。

因此，我们需要预处理 $1$ 到 $10^7$ 范围内的所有 $\mu(k)$ 值。
这与 $N=10^{14}$ 时 $\lfloor\sqrt{N}\rfloor$ 的范围（$10^7$）一致。

算法步骤：

  1. 欧拉筛：筛出 $1$ 到 $10^7$ 范围内的 $\mu$ 函数。
  2. 主循环 (计算 S(N))：
    • 令 long long N = 10^14，int S_outer = 10^7。
    • 初始化 long long total_S = 0。
    • 外层循环 $i$ 从 $1$ 到 $S_{outer}$（$i$ 必须是 long long，因为 $i^2$ 会溢出 int）。
    • 计算 long long M = N / (i * i)。
    • 计算 long long S_inner = sqrt(M)。
    • 初始化 long long f_M = 0。
    • 内层循环 $k$ 从 $1$ 到 $S_{inner}$（$k$ 必须是 long long，因为 $k^2$ 会溢出 int）。
    • 累加 $f_M \mathrel{+}= \lfloor M/k^2 \rfloor \cdot \mu[k]$。
    • 累加 total_S += (i * i) * f_M。
  3. 输出 total_S。

复杂度分析：
内层循环的计算量是 $O(\sqrt{M}) = O(\sqrt{N/i^2}) = O(\sqrt{N}/i)$。
总复杂度是 $\sum_{i=1}^{\sqrt{N}} O(\sqrt{N}/i) = O(\sqrt{N} \sum_{i=1}^{\sqrt{N}} \frac{1}{i})$。
根据调和级数，$O(\sqrt{N} \log(\sqrt{N})) = O(\sqrt{N} \log N)$，这符合你的预期。
