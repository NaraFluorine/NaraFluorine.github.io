思路分析

定义模倒数对 ((p,q)) 满足 (p 0)
]
由于在模意义下逆元通常取最小正剩余（(r < p)），可以推出 (d=1)。于是条件简化为
[
(p+q) \mid (pq+1)
]
等价地，设 (s = p+q)，则 (s \mid p^2-1)。令 (k = \frac{p^2-1}{s})，条件变为：
[
k \in \mathbb{Z}^+,\quad k \mid p^2-1,\quad k < \frac{p}{2}
]
此时 (p+q = \frac{p^2-1}{k})。每一个满足上述条件的正整数对 ((p,k)) 唯一对应一个模倒数对 ((p,q))。因此问题转化为求和：
[
F(N) = \sum{p=3}^{N} \sum{\substack{k \mid p^2-1 \ k < p/2}} \frac{p^2-1}{k}
]
直接枚举 (p) 并分解 (p^2-1) 代价过高；反过来枚举 (k) 更高效：

  • (k) 的范围为 (1 \le k \le \lfloor (N-1)/2 \rfloor)。
  • 对于每个 (k)，求所有 (x \in [0, k-1]) 满足 (x^2 \equiv 1 \pmod{k})。
  • 对每个解 (x)，(p \equiv x \pmod{k}) 且 (p > 2k)。最小 (p) 为 (x+2k)，然后以 (k) 为步长遍历到 (N)，累加 ((p^2-1)/k)。
  • 当 (k=1) 时 (p) 可取 (3) 到 (N) 的所有整数，可直接公式求和。

求 (x^2\equiv1\pmod{k}) 的解：对 (k) 进行质因数分解，针对每个素数幂求局部解（奇素数幂有 (\pm1)；(2^e) 需分类讨论），再用中国剩余定理合并。这样总解数约为 (O(N\log N))，内层循环总数约 (2\times10^7) 级别，可在 C++ 中快速完成。
